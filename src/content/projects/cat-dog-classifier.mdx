## Goal

Build a lightweight image classifier that can accurately distinguish cats from dogs on unseen images.

## Training design

- Applied image augmentation for better generalization.
- Used a CNN architecture with regularization.
- Tracked training/validation curves to avoid overfitting.

## Deployment path

The model can be served behind a simple API endpoint and consumed by a small front-end uploader for live predictions.

## Learnings

- Data quality and augmentation balance strongly affect confidence.
- Small architecture tweaks can significantly improve validation stability.
