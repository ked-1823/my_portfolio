## Goal

Use IPL historical datasets to identify trends in team outcomes, player consistency, and context-sensitive performance factors.

## Work performed

- Cleaned multi-season match and player records.
- Computed summary metrics by team, venue, and phase.
- Created visual slices to compare strike rates, wicket patterns, and win behavior.

## Findings

The analysis surfaced strong venue-specific effects and highlighted player roles with the highest match-impact consistency.

## Future enhancements

- Add predictive modeling for match outcomes.
- Build an interactive dashboard for season-by-season drill-down.
- Integrate richer contextual variables (pitch/weather proxies).
