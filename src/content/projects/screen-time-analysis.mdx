## Context

Understand user screen-time trends by day, category, and usage intensity to identify healthy vs excessive patterns.

## Analysis workflow

- Cleaned and standardized timestamped usage records.
- Generated descriptive statistics and trend breakdowns.
- Built visual summaries for category-level behavior.

## Insights

The dataset showed clear peaks in entertainment-heavy windows and highlighted opportunities for habit-aware intervention recommendations.

## Extension ideas

- Add time-series forecasting.
- Segment patterns by weekday/weekend.
- Build an interactive dashboard for exploration.
